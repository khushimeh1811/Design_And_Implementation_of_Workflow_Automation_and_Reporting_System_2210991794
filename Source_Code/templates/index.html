Workflow Automation & Audit

Upload your GOLD dataset and either a Salto or Sesam dataset, then execute the automated audit.

Salto uses a separate aggregation logic; Sesam uses a one-to-one merge on order number.
How to use
  • Upload the latest extracted dataset for GOLD and the corresponding Salto / Sesam export.
  • The system will run the audit rules and produce an .xlsx report with details + summary sheet.
  • Download the report and use it for documentation, auditing, or feeding into a reporting dashboard.
